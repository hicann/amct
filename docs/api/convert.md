# convert

## 产品支持情况

| 产品        | 是否支持 |
| ----------- | -------- |
| Ascend 950PR/Ascend 950DT | √ |
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 | √        |
| Atlas A2 训练系列产品/Atlas A2 推理系列产品 | √        |

## 功能说明

将量化校准模型转换为量化部署模型。

## 函数原型

```python
convert(model)
```

## 参数说明

| 参数名 | 输入/输出 | 说明                                                         |
| ------ | --------- | ------------------------------------------------------------ |
| model  | 输入/输出 | 输入含义：量化校准模型。<br>输出含义：量化部署模型。<br>数据类型：torch.nn.Module。 |


## 返回值说明
量化部署模型。

## 调用示例
```python
# 调用前序接口生成量化校准模型
quantize(model, cfg)
# 调用接口将量化校准模型转换为量化部署模型
convert(model)
```
