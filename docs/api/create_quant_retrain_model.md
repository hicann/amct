## 功能说明<a name="zh-cn_topic_0240187365_section15406195619561"></a>

量化感知训练接口，将输入的待量化的图结构按照给定的量化配置文件进行量化处理，在传入的图结构中插入量化相关的算子（数据和权重的量化感知训练层以及找N的层），生成量化因子记录文件record\_file，返回修改后可用于量化感知训练的torch.nn.Module模型。

## 函数原型<a name="zh-cn_topic_0240187365_zh-cn_topic_0122830089_section428121323411"></a>

```python
quant_retrain_model = create_quant_retrain_model (config_file, model, record_file, input_data)
```
